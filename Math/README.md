### 磁带文件存放优化

1. 若访问概率相同，则按照文件长度从小到大排列
2. 若文件长度相同，则按照访问概率有大到小排列
3. 若都不确定，则按照$p(i)/l(i)$从大到小排列

### 桶中取黑白球

2. 利用XOR操作，白球为1，黑球为0，最后结果为0，即剩下黑球
